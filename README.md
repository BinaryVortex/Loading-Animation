# Loading Animation

A small loading animation created using HTML and CSS.

![Preview of the loading animation](./Screenshot%202024-04-28%20232158.png)

## Demo / Preview
Open `index.html` in any modern browser to see the animation in action.

## Features
- Pure HTML & CSS (no JavaScript)
- Lightweight and easy to customize
- Cross-browser friendly with modern browsers

## How it works (brief)
The animation is implemented with CSS keyframes and transforms. The visual effect is created by animating the shapes and their opacity/position using a combination of `@keyframes`, `transform`, and timing functions.

## Files
- `index.html` — example markup and structure
- `style.css` — animation styles and keyframes
- `Screenshot 2024-04-28 232158.png` — preview image shown above

## Customization
- Colors: edit the CSS variables or color values in `style.css`.
- Size: change container dimensions or scale transform values.
- Timing: adjust `animation-duration` or `animation-delay` to change speed and rhythm.

## Usage
1. Clone or download the repository.
2. Open `index.html` in your browser.
